الذكاء الاصطناعي

الذكاء الاصطناعي هو قدرة الأجهزة على التعلم واتخاذ القرارات. يُستخدم في المساعدات الصوتية مثل Siri و Google Assistant، والروبوتات الحديثة.

💻

لغات البرمجة

لغات البرمجة مثل HTML و Python و Java تُستخدم لبناء التطبيقات والمواقع والأنظمة الذكية. كل لغة لها استخدامات مختلفة.

🌐

تصميم مواقع الويب

تصميم المواقع يتم باستخدام HTML لتكوين الصفحة، CSS لتنسيق الشكل، وJavaScript لإضافة التفاعل والحركة.
